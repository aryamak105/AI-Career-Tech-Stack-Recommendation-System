## 📌 Project Overview

The AI-Powered Tech Stack Recommender is a recommendation system that analyzes a user's raw technical skills and recommends the most suitable career paths, technology stacks, and learning roadmaps.

The system uses pattern matching and similarity scoring techniques to compare user skills with predefined job-role requirements and generate ranked recommendations.

This project was developed as part of the Artificial Intelligence Internship at DecodeLabs under Project 3: AI Recommendation Logic.

---

## 🎯 Objectives

- Collect user skills as input
- Compare user skills with industry job roles
- Calculate match percentages
- Recommend suitable career paths
- Suggest technology stacks
- Identify missing skills
- Generate a personalized learning roadmap
- Save recommendations into a report file

## 📊 Working Process

### Step 1
User enters technical skills.

Example:

```
python, sql, machine learning, pandas
```

### Step 2
System compares user skills with predefined job-role skill sets.

### Step 3
Match score is calculated.

### Step 4
Job roles are ranked based on match percentage.

### Step 5
System recommends:

- Best Career Path
- Technology Stack
- Missing Skills
- Learning Roadmap
- Expected Salary

### Step 6
A report is generated and stored in:

```
recommendations.txt
```
